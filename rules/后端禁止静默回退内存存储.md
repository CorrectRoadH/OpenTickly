# 后端禁止静默回退到内存存储

后端运行时启动必须从环境变量读取明确的数据源配置，并在缺失时立即失败。内存存储只能作为测试夹具或显式选择的开发工具，不能作为生产运行时的静默回退路径。

```grit
language go
or {
  `memory.NewStore()`,
  `memorystore.NewStore()`,
  `inmemory.NewStore()`
} where {
  $filename <: r".*apps/backend/internal/(bootstrap|platform|transport|infra|application|domain)/.*\.go",
  !$filename <: r".*_test\.go",
  !$filename <: r".*/testsupport/.*"
}
```

## 反例

```go
if databaseURL == "" {
  store = memory.NewStore()
}
```

## 正例

```go
if databaseURL == "" {
  return Config{}, errors.New("DATABASE_URL is required")
}
```
