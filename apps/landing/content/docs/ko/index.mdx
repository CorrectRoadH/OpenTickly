OpenTickly은 무료이면서, 프라이버시 우선, AI 친화적인 Toggl 대안입니다.

Toggl은 훌륭한 제품이지만 가격이 너무 비싸고 데이터가 진정한 의미에서 여러분의 것은 아닙니다. 게다가 시간당 30회라는 API 호출 제한은 골칫거리입니다. AI 에이전트라면 1분 만에 한 시간 치 할당량을 소진해 버릴 수 있습니다.

OpenTickly의 목표는 Toggl과 완전한 호환성을 유지하는 것입니다. 양쪽 사이에서 사용자 데이터를 아무런 손실 없이 옮겨 들이고 내보낼 수 있습니다. 모든 워크플로와 사용 습관도 그대로 유지됩니다. 바뀌는 것은 없습니다 — 그저 더 쓰기 편하고, 더 저렴하고, 더 프라이버시 친화적일 뿐입니다.

## 기능 개발 범위

OpenTickly의 개발 계획은 Toggl이 원래 가지고 있는 기능과, 사용자 데이터에 영향을 주지 않는 확장 기능만 고려합니다. 이렇게 하면 얻는 이점:

- 언제든 데이터를 가지고 Toggl로 돌아갈 수 있습니다 — 벤더 락인이 없습니다.

예를 들어 인스턴스 관리, AI 관련 확장 같은 것들입니다. 설령 유용하거나 환영받을 법한 개선이라 해도, 사용자 데이터가 Toggl과 분기하게 만드는 변경은 받아들이지 않습니다.
