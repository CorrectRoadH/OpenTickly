OpenToggl 是一个开源时间追踪工具，目标很简单：兼容 Toggl，让你把数据握在自己手里。

具体来说：

- 兼容 `Track API v9`、`Reports API v3`、`Webhooks API v1`
- 提供完整的 Web 界面
- 支持托管和自托管两种方式
- 支持从 Toggl 导入数据

## 为什么做这个项目

Toggl 好用，但数据不在自己手里。OpenToggl 想解决这个问题——用同样的 API，同样的体验，但你可以自己部署、自己掌控。

项目以 OpenAPI 合约和产品文档作为实现标准，确保：

- 自托管和托管版功能完全一致
- 文档、合约、代码三者保持同步
- 不做半成品发布

## 仓库结构

- `docs/` — 产品定义
- `openapi/` — OpenAPI 合约
- `apps/` — 前后端源码
- `docs/self-hosting/` — 自托管指南

## 当前范围

- 通过 Track API 进行时间追踪和工作区管理
- 通过 Reports API 查询报表
- 通过 Webhooks API 管理 Webhook
- 配套 Web 界面
- Toggl 数据导入

关于 AI：OpenToggl 不做独立的 AI 产品，但它的 API 稳定、文档齐全、支持自部署，天然适合 agent 和自动化场景。
