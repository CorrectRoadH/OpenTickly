使用 [`CorrectRoadH/toggl-cli`](https://github.com/CorrectRoadH/toggl-cli) 接入 Claude Code、OpenClaw、Hermes Agent、 Codex、Cursor 等 AI Agent，让它们直接通过命令行操作你的 OpenToggl 实例，自动记录时间、启停计时器、管理项目与标签。

- `toggl` CLI — 命令行直接操作
- `SKILL.md` 技能包 — 装进 Agent 环境一键使用

## 安装

```bash
npm install -g @correctroadh/toggl-cli

npx skills add CorrectRoadH/toggl-cli
```

## 配置

该 CLI 是同时兼容 Toggl 和 OpenToggl 的，默认指向官方 Toggl API。要改成自己的实例，认证时加上 `--api-type opentoggl` 和 `--api-url`：

不过由于 Toggl 有 30 次/小时的 API 限制，强烈建议配合 OpenToggl 使用，解锁 AI Agent 的全部潜力：

```bash
toggl auth <YOUR_API_TOKEN> --api-type opentoggl --api-url https://your-instance.com/api/v9
```

`YOUR_API_TOKEN` 可以在 OpenToggl Web UI 的个人头像 → Profile → API Token 里找到。

装好之后怎么用？看[实践 / 工作流](/zh/docs/practices)里的几个具体例子：让 Agent 帮你开计时、把周计划塞给 Agent、每晚 cron 让 AI 做时间复盘。
