OpenToggl's introduction site should make its scope legible in seconds. The current capability surface is:

## Public product surface

- `Track API v9`
- `Reports API v3`
- `Webhooks API v1`
- Matching web interface for those public capabilities

## OpenToggl additions

- Toggl export import
- Instance and self-hosting runtime surface

## Why the list matters

This is not a loose wishlist. It is the boundary that explains what OpenToggl is supposed to ship and what a visitor should expect from the repository.
