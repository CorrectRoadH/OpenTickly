Self-hosting is a first-class part of the product story.

## What that means

- you can inspect the source
- you can run the stack on your own infrastructure
- you can control upgrade timing
- you can keep the same public contract and product surface as the hosted runtime

## Runtime shape

The repository documents a single application image plus PostgreSQL and Redis dependencies. Schema management is handled through `pgschema`, and the documented compose workflow lives in the repository alongside the source.
