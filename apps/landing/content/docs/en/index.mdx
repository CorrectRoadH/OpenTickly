OpenToggl is an open-source time tracking tool. The goal is simple: be compatible with Toggl, and let you own your data.

What that means in practice:

- Compatible with `Track API v9`, `Reports API v3`, `Webhooks API v1`
- Full web interface included
- Works hosted or self-hosted
- Import your data from Toggl

## Why this project exists

Toggl is great, but your data lives on someone else's servers. OpenToggl gives you the same API and the same experience, but you can deploy it yourself.

The project uses OpenAPI contracts and product docs as the implementation standard:

- Self-hosted and hosted versions are identical
- Docs, contracts, and code stay in sync
- No half-baked releases

## Repository layout

- `docs/` — Product definitions
- `openapi/` — OpenAPI contracts
- `apps/` — Frontend and backend source
- `docs/self-hosting/` — Self-hosting guides

## Current scope

- Time tracking and workspace management via Track API
- Reporting via Reports API
- Webhook management via Webhooks API
- Web UI for all of the above
- Toggl data import

On AI: OpenToggl isn't a separate AI product. But its stable API, thorough docs, and self-hosting support make it a natural fit for agents and automation.
