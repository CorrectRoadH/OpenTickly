OpenTickly は、無料・プライバシーファースト・AI フレンドリーな Toggl の代替です。

Toggl は素晴らしいプロダクトですが、価格が高く、データが本当の意味で自分のものにはなりません。しかも 1 時間あたり 30 回という API レート制限は頭痛の種で、AI エージェントなら 1 分で 1 時間分の枠を使い切ってしまうことさえあります。

OpenTickly のゴールは Toggl との互換性を保つことです。両者の間でユーザーデータを無損失に行き来でき、ワークフローも使い勝手もそのまま維持できます。何も変わらない、ただより使いやすく、より安く、よりプライベートになるだけです。

## 機能開発のスコープ

OpenTickly の開発計画では、Toggl がもともと持っている機能と、ユーザーデータに影響しない拡張機能だけを対象にしています。こうしておくことで：

- いつでもデータを持って Toggl に戻れます — ベンダーロックインはありません。

たとえばインスタンス管理や AI 連携などの拡張がそれにあたります。どれだけ便利で好評な機能改善であっても、ユーザーデータが Toggl と分岐してしまうようなものは受け入れません。
