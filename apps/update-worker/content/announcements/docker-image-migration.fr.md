---
title: "Avis de migration de l'image Docker"
---

**Important** : l'image Docker et le projet ont été renommés.

Si vous hébergez OpenToggl (ancienne marque) vous-même, veuillez mettre à jour la référence de votre image Docker :

**Docker Hub** :

- **Ancienne** : `correctroad/opentoggl`
- **Ancienne** : `ghcr.io/CorrectRoadH/OpenToggl`
- **Nouvelle** : `correctroad/opentickly`

Les anciennes images continueront de recevoir des mises à jour pour l'instant, mais nous recommandons
de passer aux nouvelles images pour garantir l'accès à toutes les futures versions.

En savoir plus sur [GitHub](https://github.com/CorrectRoadH/OpenTickly).
